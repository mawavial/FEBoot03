Javascript Objects

Objects

Objects are used to store collections of data and more complex entities. They are used to store various keyed collections and more complex entities. An object can be created with figure brackets {…} with an optional list of properties. A property is a “key: value” pair, where key is a string (also called a “property name”), and value can be anything.

Creating an Object

There are two ways to create an object: using an object initializer or the new Object() constructor.

Using an object initializer is the most common way to create an object. The following example creates an empty object using an object initializer:

                var objectName = {};

Using the new keyword is the second most common way to create an object. The following example creates an empty object using the new keyword:

                var objectName = new Object();

Adding Properties to an Object

Properties are added to an object using dot notation. The following example adds a property to the object, objectName:

                objectName.propertyName = propertyValue;

Properties are added to an object using dot notation. The following example adds a property to the object, objectName:

                objectName['propertyName'] = propertyValue;
